# Rotas da API

## Health

- `GET /health`

## Users

- `GET /v1/users`
- `POST /v1/users`
- `PATCH /v1/users/{user_id}`
- `DELETE /v1/users/{user_id}`

## Products

- `GET /v1/products`
- `POST /v1/products`

## Orders

- `GET /v1/orders`
- `POST /v1/orders`

## Dica

Para explorar payloads e schemas sem sair do navegador, use Swagger em `/docs`.
