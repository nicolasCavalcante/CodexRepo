# App (Streamlit)

A interface Streamlit é voltada para operação e validação rápida do fluxo CRUD.

## Funcionalidades

- Criar usuários, produtos e pedidos.
- Visualizar listas em tabela.
- Exibir erro/sucesso com base no retorno da API.

## Configuração

A URL da API é controlada por `API_URL`.

Exemplo:

```bash
export API_URL=http://localhost:8000/v1
```

## Quando usar

- Onboarding de pessoas novas.
- Testes manuais simples.
- Demonstração do fluxo end-to-end.
